I chose Vue.js as my framework and included it via a CDN. Below I will explain:

How I Installed the Framework

I didn’t actually "install" Vue on my machine. Instead, I just used a <script> tag to include the Vue library from a CDN. This means no local setup was required. The code snippet: 

            <script src="https://unpkg.com/vue@3/dist/vue.global.js"></script>

How I Built This Page

1. Created a basic index.html file.
2. Added the Vue CDN link in the <head> section.
3. Defined an id="app" container where Vue controls the content.
4. Initialized a Vue application instance and bound data() properties to the template. The {{ title }} tag above is dynamically populated by Vue's reactive data.

The code that starts Vue looks like this: 

            const { createApp } = Vue

            createApp({
              data() {
                return {
                  title: 'Welcome to My Vue Page!'
                }
              }
            }).mount('#app')

Difficulties Encountered and How I Solved Them

- **Difficulty:** At first, I wasn’t sure which framework could be used directly from a CDN without complex installation steps.
Solution: After considering React and Angular, I realized Vue is the easiest to just drop in and go. React and Angular often require additional tooling to get started comfortably, but Vue is simple enough to run right in an HTML file.
